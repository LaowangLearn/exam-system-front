## 部署说明

### 前端部署
1. 进入前端目录：`cd exam-system-front`
2. 安装依赖：`npm install`
3. 构建项目：`npm run build`
4. 将构建产物部署到 Web 服务器

### 后端部署
1. 进入后端目录：`cd exam-system-backend`
2. 编译项目：`mvn clean package`
3. 启动服务：
   - Windows 系统：双击 `start.bat` 文件
   - PowerShell：运行 `./run.ps1` 脚本

## 注意事项

1. 确保前端和后端服务的配置文件中的端口号和服务地址正确匹配
2. 部署前请检查数据库连接配置是否正确
3. 首次部署时请确保数据库已初始化
4. 如有问题，请查看系统日志进行排查
