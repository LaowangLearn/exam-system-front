## 技术栈

### 后端
- Spring Boot 2.7.18
- MyBatis-Plus 3.5.4.1
- Spring Security
- H2 Database (支持达梦/金仓/PostgreSQL)
- Lombok

### 前端
- Vue 3
- Vue Router 4
- Pinia
- Element Plus
- ECharts
- Axios

## 快速启动

### 后端
```bash
cd gov-system-backend
mvn spring-boot:run
# 或 java -jar target/gov-affairs-platform-1.0.0.jar
```

### 前端
```bash
cd gov-system-frontend
npm install
npm run dev
```

## 功能模块

1. **网格化治理系统**
   - 网格单元管理（树形结构）
   - 网格事件上报、处理、评价
   - 巡检记录管理

2. **综治信访系统**
   - 信访登记、受理、分派
   - 办理工单管理
   - 满意度评价

3. **智慧安防系统**
   - 设备管理（摄像头、门禁、报警等）
   - 实时告警
   - AI分析支持

4. **公文机要系统**
   - 公文起草、提交
   - 审批流程
   - 发布、归档

## 政务信创适配

- 支持国密SM2/SM3/SM4加密
- 等保三级合规设计
- 达梦、人大金仓、华为高斯数据库兼容
- 国产中间件适配
